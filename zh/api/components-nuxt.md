# &lt;nuxt&gt; 组件

> 该组件只适用于在[布局](/guide/views#布局)中显示页面组件（即非布局内容）。

例如 (`layouts/default.vue`)：

```html
<template>
  <div>
    <div>页头</div>
    <nuxt/>
    <div>页脚</div>
  </div>
</template>
```

可以看下这个实际的[布局示例](/examples/layouts)。
